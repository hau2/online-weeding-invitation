**`.claude/`:**
- Purpose: Get Shit Done (GSD) automation framework configuration
- Contains: Agent definitions, command specifications, workflow templates, hooks

**`.planning/`:**
- Purpose: Project planning and documentation
- Contains: Codebase analysis documents (ARCHITECTURE.md, STRUCTURE.md, etc.)
- Key files: `.planning/codebase/` - analysis documents generated by GSD mapper

**`.git/`:**
- Purpose: Git repository metadata
- Contains: Git configuration and commit history
