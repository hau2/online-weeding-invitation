    - unpublish: sets status='draft', does NOT clear slug, returns mapped invitation

       - Add `unpublish(userId, id)`: findOne to verify ownership, update status='draft' (do NOT clear slug), return mapRow
       - For the snake_case mapping in update: build an object mapping camelCase DTO keys to snake_case DB columns. Only include keys that are present (not undefined) in the dto.
