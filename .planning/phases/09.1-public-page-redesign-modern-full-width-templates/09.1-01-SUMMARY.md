## Decisions Made
- ThemeConfig uses Tailwind class strings directly (e.g., `font-black`, `rounded-xl`) for seamless className composition in JSX
- getTheme() falls back to `modern-red` for any unknown template ID (safe default)
- Plus Jakarta Sans uses `latin-ext` subset (covers Vietnamese diacritics) since the font does not offer a dedicated `vietnamese` subset like Be Vietnam Pro does
- Theme config files export named constants (e.g., `modernRed`, `minimalistBw`) imported into a central THEMES record
