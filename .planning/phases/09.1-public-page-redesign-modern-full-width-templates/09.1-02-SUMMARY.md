## Decisions Made
- CountdownSection renders two visual modes based on theme.navStyle: card-box with shadow/border for colored themes, bare large numbers with tabular-nums for mono (B&W Minimalist) -- matches both Stitch reference HTML files
- StickyNav brand text changes per navStyle: "Thiep Cuoi Online" for colored, "The Wedding" for mono -- matching Stitch screen1.html vs screen6.html patterns
- VenueSection shows all available venues (groom and/or bride) with a border separator between them, supporting the upstream side-filtering pattern
- BankQrSection wraps QR images in next/image with object-contain for consistent rendering, and checks both bankQrUrl and bankAccountNumber for card visibility
- FooterSection uses hex opacity suffixes (CC for 80%, 80 for 50%, 66 for 40%) for footer text color variations instead of separate theme properties
