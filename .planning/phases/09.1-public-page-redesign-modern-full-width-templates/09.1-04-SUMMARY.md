- Desktop mockup includes browser chrome bar (3 colored dots + address bar) for clear visual distinction from phone view
- Mobile/tablet editor shows only simple bordered preview (no mockup frames) since the editor itself is on a small screen
- Plus Jakarta Sans font variable applied to all preview containers for accurate public page font rendering in editor
