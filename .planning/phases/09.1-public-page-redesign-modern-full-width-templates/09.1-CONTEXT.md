# Phase 9.1: Public Page Redesign - Modern Full-Width Templates - Context

**Gathered:** 2026-03-16
**Status:** Ready for planning

<domain>
## Phase Boundary

Replace the current narrow 420px card-style templates (Traditional/Modern/Minimalist) with 6 modern full-width templates matching Stitch AI reference designs. Build a shared layout component with theme config system that renders all sections (hero, sticky nav, countdown, timeline, photo grid, venue/map card, bank QR cards, footer). Architecture supports admin-created themes in the future. Update editor preview to show both mobile and desktop views. Remove DesktopFrame botanical decorations.

</domain>

<decisions>
## Implementation Decisions

### Envelope + Full-Width Layout
- Envelope animation stays as-is (centered ~400px with wax seal, existing animation)
- After reveal, invitation expands/transitions into the new full-width layout
- Falling petals continue over the full-width layout (template-specific colors)
- DesktopFrame (botanical corners/vines from Phase 9) removed entirely — new full-width templates ARE the desktop experience

### Parent Names
- Parent names (Nha trai / Nha gai) displayed in the hero section, below couple names
- Both families' parents always visible regardless of ?side= parameter (existing Vietnamese cultural norm)

### Theme Architecture
- One shared React layout component renders all sections; theme config object defines visual properties
- Theme config includes: primary color, background, border-radius, font weight, hero overlay, gallery effect, petal colors
- 6 built-in theme configs hardcoded; architecture supports admin creating new themes via DB in future
- Old 3 templates (Traditional/Modern/Minimalist) replaced entirely — not kept alongside

### 6 Built-In Themes (from Stitch AI reference)
1. **Modern Red** — primary #ec1349, light bg #f8f6f6, rounded corners, bold fonts
2. **Soft Pink/Peach** — peach/coral primary, pink gradient hero, softer rounded sections
3. **Brown/Gold** — warm brown/gold primary, cream/beige bg, elegant serif feel
4. **Olive/Green** — sage green primary, warm beige bg, nature-inspired
5. **B&W Minimalist** — primary #171717, white bg, zero border-radius, grayscale-to-color gallery, no petals
6. **Classic Red/Gold** — deep red/burgundy primary, cream bg with gold ornamental accents

### Backward Compatibility
- Old templateId values auto-mapped to new themes in code (no DB migration):
  - 'traditional' -> 'classic-red-gold'
  - 'modern' -> 'modern-red'
  - 'minimalist' -> 'minimalist-bw'
- Published invitations automatically show new full-width design — no user action needed

### Hero Section
- First photo from photoUrls[] used as hero background with gradient overlay
- Shows: "Save The Date" label, ceremony title, couple names with heart, parent names, wedding date, CTA buttons
- Min-height 560px on desktop, responsive on mobile

### Countdown Timer
- Card box style (not flip-clock): large numbers in bordered/shadowed card boxes, 4-column grid
- Labels: Ngay, Gio, Phut, Giay
- Integrated into template layout (no longer separate component in InvitationShell)

### Sticky Navigation
- Desktop only (hidden on mobile) — sticky top with backdrop-blur
- Section links: Cap doi, Su kien, Album, Ban do, Mung cuoi
- Logo/brand on left, CTA "Gui loi chuc" button on right
- Mobile users scroll naturally

### Photo Gallery
- 2x3 grid (grid-cols-2 md:grid-cols-3), aspect-[3/4] portrait photos
- Hover zoom effect only (group-hover:scale-110) — no lightbox modal
- Rounded corners per theme (xl for most, 0 for B&W Minimalist)

### Venue/Map Section
- Split layout: venue info on left, static map image on right
- "Chi duong" button links to venueMapUrl
- Map image placeholder with bouncing location pin
- Shows both groom and bride venue info (filtered by ?side= param)

### Bank QR Section
- Rich cards with decorative circle backgrounds (bg-primary/5)
- 2-column grid for groom/bride sides
- Structured info: bank name, QR image, account holder, account number with copy button
- Free tier: BankQrLock still applies (blur + lock overlay)

### Footer
- Full-width with couple names, thank-you message, copyright text
- Theme-specific background (dark for B&W, light for others)

### Editor Preview Update
- Editor preview shows both mobile and desktop view mockups of the new full-width templates
- Design follows stitch-editor-page.md reference (Stitch AI editor screens)

### Claude's Discretion
- Exact CSS animation for envelope-to-fullwidth transition
- Static map image source and fallback (Google Static Maps API or placeholder)
- Exact theme config property names and TypeScript interface
- Bundle splitting strategy for the shared layout component
- Love story timeline icon mapping per theme
- Footer copyright text format
- Mobile responsive breakpoints for each section

</decisions>
